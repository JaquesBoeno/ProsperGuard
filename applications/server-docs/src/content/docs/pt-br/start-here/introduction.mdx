Esta é uma documentação feita especificamente para o back-end da aplicação ProsperGuard, portanto aqui trataremos de coisas como rotas da aplicação, quais parametros utilizar, documentação das funções principais do código entre outros.

## Tecnologias Utilizadas
- **Golang**: Neste projeto nós utilizamos a linguagem Golang para o desenvolvimento do back-end
- **Fiber**: Como framework http nós escolhemos o fiber pois ele utiliza o fasthttp como base e trás uma combinação de maior performace e facilidade no desenvolvimento.
- **Ent**: O Ent foi escolido como ORM por sua facilidade na utilização e um sistema de schemas em conjunto com as migrations.
- **Postgres**: O banco de dados Postgres foi escolhido por ser um banco já consolidado no mercado e se adequar aos principios ACID necessários neste projeto.
